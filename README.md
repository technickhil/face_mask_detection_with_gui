#  Face Mask Detection With GUI


# A very Simple Gui app for Face Mask

  - Collect Face Data
  - Build Face Classifier 
  - Detect the face
  - using tensorflow to create mask detection model

  
  
# Installation

- Install the requirements

- tensorflow>=1.15.2
- keras==2.3.1
- imutils==0.5.3
- numpy==1.18.2
- opencv-python==4.2.0.*
- matplotlib==3.2.1
- scipy==1.4.1
